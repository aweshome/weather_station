# Weather station

## Internal unit (terminal)

### Heart:
	ATxmega128A3U - uC
	
### Peripherals:
	nRF24L01 - wireless transceiver
	PMSA003 - dust sensor
	AM2302 - temperature and humidity sensor

## External unit (sensor)

### Heart:
	ATmega1284
	
### Peripherals:
	EG9018C - 640x480 8.4" mono lcd screen
	DS3231 - rtc
	BMP180 - pressure sensor
	nRF24L01
	AM2302
